HAND GESTURE RECOGNITION SYSTEM - FRONTEND

OVERVIEW
This frontend provides a user interface to upload hand gestures
and display predicted results along with confidence values.

FEATURES
- Gesture upload interface
- Prediction result display
- Confidence visualization
- Simple and responsive UI

FOLDER STRUCTURE
frontend/
- src/
  - components/    Reusable UI components
  - pages/         Application pages
  - utils/         Helper and API functions
- package.json     Frontend dependencies
- vite.config.js   Build configuration

INSTALLATION
Open terminal inside frontend folder and run:
npm install

RUN FRONTEND
npm run dev

TECHNOLOGIES USED
React
Vite
JavaScript
HTML
CSS
Chart.js
